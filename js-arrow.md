# JS Arrow

Arrow functions allow us to write shorter function syntax:

```js
let myFunction = (a, b) => a * b;
```

- without arrow:

    ```js
    hello = function() {
        return "Hello world!";
    }
    ```

- with arrow:

    ```js
    hello = () => {
        return "Hello World!";
    }
    ```

    ```js
    // when only one statements
    hello = () => "Hello World!";
    ```
